# Mod Library

A free, searchable library of mods with links to the original creators.

## Features
- Search by mod name or creator
- Filter by category
- Download/info links point to original creators

## How to update
1. Edit `mods.json` to add/remove mods
2. Update `index.html`, `styles.css`, or `script.js` if needed
3. Upload the updated files to GitHub and commit changes
